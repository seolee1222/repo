NMS(Network Management System) Zabbix구축

🖥️ NMS 네트워크 관리 시스템

이번 포스트에서는 Network Management System(NMS)인 Zabbix를 구축하는 방법에 대해 자세히 알아보겠습니다.

📋 Zabbix란?

Zabbix는 NMS(Network Management System)네트워크 관리 프로그램으로 서버, 네트워크 장비, 애플리케이션의 성능을 실시간으로 모니터링할 수 있는 강력한 도구입니다.

🚀 주요 특징

  • 실시간 모니터링: 네트워크 장비와 서버의 상태를 실시간으로 감시
  • 알림 시스템: 문제 발생 시 이메일, SMS 등으로 즉시 알림
  • 데이터 시각화: 그래프와 대시보드를 통한 직관적인 데이터 표현
  • 자동 검색: 네트워크 장비 자동 검색 및 등록

🛠️ 설치 환경

이번 실습에서는 다음과 같은 환경에서 Zabbix를 설치하겠습니다:

  • OS: Ubuntu 22.04 LTS
  • Database: MARIADB
  • Web Server: Apache 2
  • PHP: 8.3

📥 설치 과정

1. 패키지 업데이트

sudo apt update && sudo apt upgrade -y

2. MySQL 설치 및 설정

sudo apt install mysql-server -y
sudo mysql_secure_installation

3. Apache 및 PHP 설치

sudo apt install apache2 php php-mysql php-gd php-xml php-bcmath php-mbstring -y

4. Zabbix 저장소 추가

wget https://repo.zabbix.com/zabbix/6.4/ubuntu/pool/main/z/zabbix-release/zabbix-release_6.4-1+ubuntu22.04_all.deb
sudo dpkg -i zabbix-release_6.4-1+ubuntu22.04_all.deb
sudo apt update

5. Zabbix 서버 설치

sudo apt install zabbix-server-mysql zabbix-frontend-php zabbix-apache-conf zabbix-sql-scripts zabbix-agent -y

🔧 데이터베이스 설정

MySQL에서 Zabbix 전용 데이터베이스를 생성하고 초기 스키마를 가져옵니다:

mysql -u root -p

CREATE DATABASE zabbix CHARACTER SET utf8mb4 COLLATE utf8mb4_bin; // 자빅스 데이터 베이스에 chaeacter set utf8로 하겠다
CREATE USER 'zabbix'@'localhost' IDENTIFIED BY 'password'; // 유저생성 및 비밀번호주기
GRANT ALL PRIVILEGES ON zabbix.* TO 'zabbix'@'localhost'; //모든권한을 주겠다
FLUSH PRIVILEGES;
EXIT;

📊 웹 인터페이스 설정

Zabbix 웹 인터페이스에 접속하여 초기 설정을 완료합니다. 브라우저에서 http://your-server-ip/zabbix로 접속하면 설정 마법사가 시작됩니다.
